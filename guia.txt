1. ?QUÉ ES IVAN RESELLER WEB?
================================================================================

Ivan Reseller Web es una plataforma completa de dropshipping automatizado que
permite generar ingresos de forma pasiva mediante la venta de productos en
múltiples marketplaces (eBay, Amazon, MercadoLibre) sin necesidad de tener
inventario físico.

EL SISTEMA FUNCIONA ASÍ:
- Busca productos rentables en AliExpress (proveedor chino)
- Analiza la competencia en eBay, Amazon y MercadoLibre
- Calcula automáticamente precios, márgenes y rentabilidad
- Publica productos en los marketplaces seleccionados
- Cuando alguien compra, el sistema procesa la orden automáticamente
- El usuario gana la diferencia entre el precio de venta y el costo

TODO ESTO FUNCIONA 24/7 SIN INTERVENCIÓN HUMANA cuando está configurado
correctamente.

2. CÓMO GENERA DINERO AUTOMÁTICAMENTE
================================================================================

FLUJO COMPLETO DE GENERACIÓN DE DINERO:

PASO 1: BÚSQUEDA AUTOMÁTICA DE OPORTUNIDADES
---------------------------------------------
El sistema busca productos en AliExpress que cumplan estos criterios:
- Margen de ganancia mínimo: 10% (configurable)
- ROI mínimo: 50% (configurable)
- Ganancia mínima: $10 USD (configurable)
- Demanda validada con Google Trends
- Competencia analizada en tiempo real

PASO 2: ANÁLISIS INTELIGENTE
-----------------------------
Para cada producto encontrado, el sistema:
- Calcula el precio de venta óptimo considerando:
  * Costo del producto en AliExpress
  * Costo de envío
  * Impuestos de importación
  * Comisiones del marketplace (eBay, Amazon, MercadoLibre)
  * Comisiones del sistema (10% + costo fijo mensual)
  * Márgenes de seguridad
- Analiza la competencia en cada marketplace
- Valida la demanda real con Google Trends
- Calcula tiempo estimado hasta primera venta
- Calcula tiempo hasta recuperar inversión (break-even)

PASO 3: PUBLICACIÓN AUTOMÁTICA
-------------------------------
El sistema publica el producto en los marketplaces configurados:
- Crea el listing con título optimizado
- Sube todas las imágenes del producto
- Escribe descripción profesional
- Establece precio calculado automáticamente
- Configura categoría correcta
- Gestiona inventario (cantidad disponible)

PASO 4: RECEPCIÓN DE VENTAS
----------------------------
Cuando un cliente compra en eBay/Amazon/MercadoLibre:
- El sistema recibe notificación automática vía webhook
- Crea registro de venta en la base de datos
- Calcula comisión automáticamente (10% + costo fijo)
- Notifica al usuario en tiempo real
- Reserva el capital necesario para comprar el producto

PASO 5: PROCESAMIENTO AUTOMÁTICO DE ÓRDENES
--------------------------------------------
El sistema puede (si está configurado):
- Comprar automáticamente el producto en AliExpress
- Usar el link de afiliado para ganar comisión adicional
- Enviar el producto directamente al cliente
- Actualizar el estado del pedido
- Gestionar reembolsos y devoluciones

GANANCIA DEL USUARIO:
---------------------
Ganancia = Precio de Venta - Costo del Producto - Envío - Impuestos - 
           Comisiones Marketplace - Comisión Sistema

Ejemplo práctico:
- Producto en AliExpress: $10 USD
- Precio de venta calculado: $25 USD
- Comisión eBay (10%): $2.50
- Comisión sistema (10%): $2.50
- Envío e impuestos: $3
- GANANCIA NETA: $7 USD por venta

Si vendes 10 productos al día: $70 USD/día = $2,100 USD/mes

3. FUNCIONALIDADES PRINCIPALES

3.1. BÚSQUEDA Y ANÁLISIS DE OPORTUNIDADES
------------------------------------------
- Búsqueda manual: Ingresa términos de búsqueda y el sistema encuentra

- Búsqueda automática: El sistema busca continuamente nuevas oportunidades
- Análisis de competencia: Compara precios en eBay, Amazon y MercadoLibre
- Cálculo de rentabilidad: ROI, margen, ganancia neta automáticos

  búsquedas reales

3.2. SCRAPING AVANZADO DE PRODUCTOS
------------------------------------
- Scraping stealth: Usa Puppeteer con plugins anti-detección
- Rotación de proxies: 50+ proxies para evitar bloqueos

- Extracción completa: Título, precio, imágenes, descripción, especificaciones,
  reviews, rating, vendedor

4. SISTEMAS DE AUTOMATIZACIÓN
================================================================================

4.1. SISTEMA AUTOPILOT (AUTÓNOMO 24/7)
--------------------------------------
El sistema más potente: opera completamente solo.

FUNCIONAMIENTO:
- Se ejecuta en ciclos configurables (por defecto cada 60 minutos)
- En cada ciclo:
  1. Selecciona término de búsqueda inteligentemente (basado en performance)
  2. Busca oportunidades en AliExpress
  3. Analiza competencia en marketplaces
  4. Calcula rentabilidad
  5. Filtra por reglas de negocio (margen mínimo, ROI mínimo, capital disponible)
  6. Publica automáticamente O envía a aprobación (según configuración)
  7. Actualiza métricas y aprende de los resultados

CONFIGURACIÓN:
- Habilitado/Deshabilitado
- Intervalo entre ciclos (minutos)
- Modo de publicación: Automático o Manual (requiere aprobación)
- Marketplace objetivo: eBay, Amazon, MercadoLibre
- Máximo de oportunidades por ciclo
- Términos de búsqueda personalizados
- Capital de trabajo disponible
- Ganancia mínima (USD)
- ROI mínimo (%)
- Optimización habilitada/deshabilitada

OPTIMIZACIÓN INTELIGENTE:
- Aprende qué categorías funcionan mejor
- Prioriza búsquedas con mejor performance histórica
- Ajusta estrategia basado en resultados
- 80% explotación (categorías probadas), 20% exploración (nuevas)

GESTIÓN DE CAPITAL:
- Rastrea capital disponible en tiempo real
- Reserva capital automáticamente por producto
- Previene sobre-compromiso
- Cálculo: Capital Total - Pedidos Pendientes - Productos Aprobados

MÉTRICAS Y REPORTES:

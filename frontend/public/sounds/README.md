# Sale notification sound

The file `sonido_venta.mp3` is played when a new sale notification is received (if the app is open and the user has granted notification permission).

- A minimal placeholder file is included so the path works; you can replace it with a short (1–3 s) sound of your choice (e.g. cash register or ding).
- Royalty-free sources: [BigSoundBank](https://bigsoundbank.com) (CC0), [Freesound](https://freesound.org), or similar. Save the file as `sonido_venta.mp3` in this folder.
- If the file is missing or fails to play, the app falls back to a short beep generated in the browser.

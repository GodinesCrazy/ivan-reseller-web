# Post-Sale Real-Profit Criteria

Date: 2026-03-21
Status: Persistent truth rule

## Mandatory Distinctions

- listing published != sale completed
- order ingested != supplier fulfilled
- supplier order created != supplier payment completed
- delivered state != released funds
- projected margin != realized profit

## Required Proof Chain

1. valid ML Chile order
2. commercial-proof eligibility remains true
3. supplier purchase executed against correct AliExpress item
4. supplier payment completed
5. tracking becomes real
6. buyer receives item
7. MercadoLibre releases funds
8. realized profit is recognized

## Forbidden Shortcuts

- counting cancelled orders as success
- counting payout-failed sales as success
- counting supplier-order creation as fulfillment completion
- counting demo or synthetic rows as commercial proof

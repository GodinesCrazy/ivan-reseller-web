## Discover flow: how it connects to the pipeline

```
Discover search ‚Üí user selects product
  ‚Üí POST /discover/evaluate     (optional: view pricing breakdown live)
  ‚Üí POST /discover/import-draft (creates DB snapshot + draft listing)
  ‚Üí navigate to /cj-shopify-usa/listings
  ‚Üí POST /listings/publish       (operator publishes draft to Shopify)
```

The `import-draft` call:
1. Calls `adapter.getProductById()` ‚Äî fetches CJ product with all variants
2. Upserts `CjShopifyUsaProduct` + all `CjShopifyUsaProductVariant` records
3. Calls `adapter.quoteShippingToUsWarehouseAware()` ‚Äî saves a `CjShopifyUsaShippingQuote`
4. Calls `cjShopifyUsaPublishService.buildDraft()` using the new DB product ID
5. Returns listing ID + suggested price

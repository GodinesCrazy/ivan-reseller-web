# P103 — Competitive / marketplace pattern notes

Mature multichannel listing tools and marketplace programs converge on:

1. **Main image = plain product on neutral background** — Amazon-style “pure white” (#RGB 255) for many catalogs; Mercado Libre Chile enforces **white / light** main photo in practice (Seller Center copy: “fondo blanco”).
2. **No marketing chrome on the main photo** — text, logos, badges, watermarks, and “promo card” layouts are rejected or down-ranked.
3. **Conservative source selection** — operators pick the **cleanest** supplier frame (single product, even lighting, minimal clutter) *before* heavy editing; padding a bad frame wastes moderation budget.
4. **Separation before resize** — background removal (manual or automated) before squaring reduces halo complaints vs flattening busy scenes.

P103 aligns automation with (3)+(4): **rank → isolate → white hero**, rather than **re-pad the last failed portada**.
